# Deimos-AreaBan
Unicode Character Page Generator for Minecraft Java Areabans 1.21+

Generates 100 pages each containing 1023 unreversible Unicode characters that cannot be compressed in order to reach the Java packet size limit.

![alt text](https://github.com/Titanite01/Deimos-AreaBan/blob/main/icon2.png)

Open in any default python compiler to generate the codes.
